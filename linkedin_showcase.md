# LinkedIn Showcase: Video Script & Post Draft

Ye guide aapko LinkedIn par apna kaam dikhane (showcase) mein madad karegi.

---

## 🎬 1. Video Script (Demo Video ke liye)

Aap apne screen ko record karte hue (jaise OBS ya Loom ka use karke) ye steps follow karein:

**Scene 1: Introduction (10-15 seconds)**
- *Voiceover:* "Hello everyone! Maine hal hi mein Cognifyz Technologies ke saath apni Machine Learning internship complete ki hai. Aaj main aapko apne projects ka ek quick demo dikhaunga."
- *Screen:* Apna `main.py` dikhayein.

**Scene 2: Task 1 - Rating Prediction (20 seconds)**
- *Voiceover:* "Mera pehla task tha linear aur decision tree regression ka use karke restaurant ratings predict karna. Mere model ne 92% accuracy achieve ki."
- *Screen:* `predict_ratings.py` ko run karein aur plots (feature importance) dikhayein.

**Scene 3: Task 2 - Cuisine Classification (20 seconds)**
- *Voiceover:* "Doosre task mein maine Random Forest classifier ka use karke restaurants ki cuisines classify ki. Ismein maine top 20 cuisines aur unke imbalance ko handle kiya."
- *Screen:* `classify_cuisines.py` ko run karein aur confusion matrix heatmap dikhayein.

**Scene 4: Task 3 - Recommendation System (20 seconds)**
- *Voiceover:* "Maine ek content-based recommendation system bhi banaya jo TF-IDF aur Cosine Similarity ka use karke users ko unki pasand ke restaurants suggest karta hai."
- *Screen:* `recommend_restaurants.py` run karke kuch recommendations dikhayein.

**Scene 5: Task 4 - Location Analysis (20 seconds)**
- *Voiceover:* "Aakhir mein maine geographical analysis kiya, jahan maine sheharon ke hisaab se ratings aur price patterns ko analyze kiya."
- *Screen:* `location_analysis.py` ke plots (maps) dikhayein.

**Scene 6: Conclusion (10 seconds)**
- *Voiceover:* "Ye internship mere liye ek bahut bada learning experience rahi. Thank you Cognifyz Technologies is opportunity ke liye!"
